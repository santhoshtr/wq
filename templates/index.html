An experimental natural language based querying system for Wikipedia. Questions are generated using gpt-3.5-turbo model by OpenAI. Answering questions is based on distilbert-base-cased-distilled-squad model optimized using ONNX. The answering context is provided using a hybrid strategy of named entity recognition on query and titles extracted from Wikipedia native search results for the query.
